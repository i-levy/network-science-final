# Spring 2026 DS8104 - Network Science Final Project Repo

## Replicating and extending [*Small-World Propensity and Weighted Brain Networks*](https://doi.org/10.1038/srep22057)
This project will involve re-writing the Small-World Propensity (SWP) metric in `Python`, then replicating the results for *C. elegans* as described in the paper, using the original `MATLAB` implementation of SWP to confirm the validity.

To extend this concept, I will apply SWP to 14 different measures of functional connectivity, calculated on resting-state fMRI from the Human Connectome Project using the *sonnet* subset of measures from [`pyspi`](https://time-series-features.gitbook.io/pyspi).
